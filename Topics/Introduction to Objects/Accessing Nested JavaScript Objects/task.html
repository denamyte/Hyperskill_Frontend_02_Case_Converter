Accessing Nested JavaScript Objects

Given the following JavaScript object: 

let school = {
  name: 'ABC High School',
  students: [
    {
      name: 'John',
      age: 20,
      courses: ['Math', 'Physics']
    },
    {
      name: 'Jane',
      age: 19,
      courses: ['English', 'Chemistry']
    }
  ]
}
Which of the following lines of code correctly accesses the name of the school, the name of the first student and the first course they are taking?
